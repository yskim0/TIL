## YOLO labeling 설정
* labelimg 사용 
    * [labelimg github](https://github.com/tzutalin/labelImg) 
* 환경 설정
    * Python 3 Virtualenv (Recommended) 로 시도했다가 실패하고 
    anaconda 환경 설정을 그냥 새로 만듦.
    * 가상환경 activate 시키고 Python 3 + Qt5 (Recommended) 시키는 대로 함.  
     **주의: pip로 깔 것**
    * git clone 해둔 장소로 가서 아래 코드 실행
    ```
    make qt5py3
    python3 labelImg.py
    ```

* 이제 라벨링 노가다를 향하여 📈
